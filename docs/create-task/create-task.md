# 作成
1. 日本エリアを選択してください。
2. 正確な SKU コードを使用してください。
3. サイズが空の場合、アプリはランダム サイズを使用します。<br/> 
サイズを入力するときは、JP25、JP27などのJPサイズ値を使用してください.<br/>
「JP」と数字の間にスペースはありません。<br/>
4. アカウント、プロファイル、およびプロキシ グループを選択してください。

# その他設定

- 設定ページでは、Webhook を設定できます。
- Number of Browser Instance: バックグラウンド プロセスで実行されているブラウザの数です。

  これは RAM サイズに依存します。<br/>
  RAM 8GB: 3~5 <br/>
  RAM 16GB: 3~7<br/>
  RAM 32GB: 3~10<br/>
